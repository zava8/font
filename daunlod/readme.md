# hinglish.ttf or hnskrit.ttf ko linux vindoz me daunlod krke install:

1. hm github.com/zava8 font repo me zaye
2. unicase folder me zakr hinglish.ttf hnskrit.ttf daunlod kre.
3. font zis foldr me h use right krke root folder ki Trh khole.
4. font file ko right klik kre -> font manager -> install button klik kre.
5. font file ko right klik kre -> fonts -> install button klik kre.
6. ~/.local/share/fonts/ me font files (hinglish.ttf hnskrit.ttf) ko paste kre.
7. /usr/local/share/fonts/ me font files (hinglish.ttf hnskrit.ttf) ko paste kre.
8. /home/viml/.fonts/ me font files (hinglish.ttf hnskrit.ttf) ko paste kre.
9. /usr/share/fonts/truetype me hscii foldr bnaye.
L. /usr/share/fonts/truetype/hscii/ me font files (hinglish.ttf hnskrit.ttf) ko paste kre.

J. Androed mobile me install krne ke liye **zfont 3 app** install kre , fir (hinglish.ttf hnskrit.ttf) ko daunlod kre.
Q. hinglish.ttf ko zfont 3 ke sath khole or old method se install kre. reboot ke liye puche toh ha kre.
